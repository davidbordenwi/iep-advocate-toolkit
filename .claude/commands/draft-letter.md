# Draft Letter to District

Help draft a formal letter to the school district.

## Instructions

1. **Read STUDENT.md** for student information to include in the letter

2. **Ask what type of letter is needed:**
   - Request for evaluation/re-evaluation
   - Prior Written Notice request
   - Concern about services/implementation
   - Request for IEP meeting
   - Response to district communication
   - Other formal request

3. **Gather necessary details:**
   - Recipient (if known)
   - Specific concern or request
   - Timeline/deadlines
   - Any previous correspondence to reference

4. **Draft the letter** following these guidelines:
   - Use formal, professional tone
   - Reference specific legal requirements when appropriate
   - Include specific dates, data points, and documentation
   - Request written response/Prior Written Notice
   - Keep collaborative tone while being firm

5. **Save to correspondence/** folder with appropriate naming (YYYY-MM-DD-description.md)

## Student Information

Reference STUDENT.md for:
- Student name
- Date of birth
- Grade
- District name
- Eligibility category
