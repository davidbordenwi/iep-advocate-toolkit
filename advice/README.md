# Advice Library

Guides for common situations and procedures.

## Folders

- `what-if/` - Situation-specific guidance (e.g., "What if they deny my request?")
- `how-to/` - Step-by-step procedures (e.g., "How to request an IEE")

## Quick Access

Use `/what-if` command to get situation-based advice.

## Structure

Each topic has:
- **Quick Card** - One-page reference for quick answers
- **Detailed Guide** - In-depth explanation with examples
