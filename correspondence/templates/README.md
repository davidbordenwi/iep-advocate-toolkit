# Correspondence Templates

Ready-to-use letter and email templates.

## Available Templates

Create templates for common situations:
- Request for evaluation
- Request for IEP meeting
- Follow-up after meeting
- Prior Written Notice request
- Disagreement with decision
- Record request (FERPA)

## Using Templates

1. Copy the template you need
2. Fill in the bracketed placeholders
3. Review and personalize
4. Send and log in correspondence-log.md
