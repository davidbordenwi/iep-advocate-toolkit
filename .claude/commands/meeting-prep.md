# Meeting Preparation

Help prepare for an upcoming IEP meeting.

## Instructions

1. **Read context files:**
   - STUDENT.md - for student profile and key data
   - CURRENT-STATUS.md - for current situation
   - MEMORY.md - for important context

2. **Ask for meeting details** (if not provided):
   - Meeting date and time
   - Meeting purpose/type

3. **Review relevant documents:**
   - evaluations/ folder
   - iep-documents/ folder
   - Recent correspondence

4. **Create a preparation document** that includes:
   - Meeting objectives (what we want to accomplish)
   - Key data points to reference (from evaluations)
   - Specific requests/concerns to raise
   - Questions to ask the team
   - Supporting evidence citations

5. **Save the prep document:**
   - To meetings/[date]/ folder for the specific meeting
   - Or meetings/templates/ if it's a reusable template

## Key Data Points

Reference the Key Data Points section from STUDENT.md for specific scores and metrics.

## Tone Guidance

- Maintain collaborative approach
- Focus on student's needs, not blame
- Reference specific data and evaluations
- Frame requests in terms of FAPE (Free Appropriate Public Education)
