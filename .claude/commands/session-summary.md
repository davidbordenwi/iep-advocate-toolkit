# Session Summary

Log this advocacy session and capture learnings.

## Instructions

1. **Review what was discussed/accomplished** in this session

2. **Create session log** in sessions/:
   ```markdown
   # Session: [Date]

   ## Summary
   [Brief overview of what was covered]

   ## Tasks Completed
   - [Task 1]
   - [Task 2]

   ## Decisions Made
   - [Decision and reasoning]

   ## Documents Created/Updated
   - [File path]

   ## New Information Learned
   - [Important fact or update]

   ## Action Items (for later)
   - [ ] [Action item]

   ## Questions to Research
   - [Question]

   ## Emotional Check-in
   [Optional: How is the family feeling about advocacy progress?]
   ```

3. **Update other files if needed:**
   - MEMORY.md - Add important facts to remember
   - CURRENT-STATUS.md - Update if status changed
   - lessons-learned.md - Add advocacy insights
   - calendar/deadlines.md - Add new deadlines

4. **Ask if anything should be:**
   - Added to MEMORY.md
   - Added to lessons-learned.md
   - Flagged for follow-up

5. **Remind about pending action items** from this session

## At Session End

"Before we wrap up:
- [Summarize key decisions]
- [List action items for the family]
- [Note any upcoming deadlines]

Is there anything else you'd like me to remember for next time?"
