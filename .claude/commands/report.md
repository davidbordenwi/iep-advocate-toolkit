# Generate Report

Generate progress reports and advocacy summaries.

## Instructions

1. **Ask what type of report is needed:**
   - Progress summary (for a time period)
   - Meeting summary
   - Advocacy status report
   - Service tracking report
   - Communication audit
   - Custom report

2. **Gather date range or scope** (if applicable)

3. **Collect data from relevant files:**
   - timeline/advocacy-log.md
   - progress/ folder
   - correspondence/correspondence-log.md
   - calendar/deadlines.md
   - evaluations/
   - iep-documents/

4. **Generate the report:**

### Progress Summary Format
```markdown
# Progress Summary: [Date Range]

## Overview
[Brief summary of the period]

## Key Events
- [Date]: [Event]
- [Date]: [Event]

## Wins
- [Accomplishment]

## Ongoing Concerns
- [Issue]

## Services Status
| Service | Status | Notes |
|---------|--------|-------|
| [Service] | [On track/Concern/Not started] | [Details] |

## Next Steps
- [ ] [Action item]

## Upcoming Deadlines
- [Deadline]: [Date]
```

5. **Save to reports/generated/** with descriptive name

6. **Offer to:**
   - Email the report
   - Print-friendly format
   - Create action items from findings
