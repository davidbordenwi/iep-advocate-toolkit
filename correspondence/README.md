# Correspondence

Track all communications with the school here.

## Contents

- `correspondence-log.md` - Running log of all communications
- `templates/` - Letter and email templates

## Using the Correspondence Log

Use `/log-correspondence` to quickly add entries, or add manually:

```markdown
### YYYY-MM-DD - Type
**From/To:** [Names]
**Method:** Email/Letter/Phone
**Subject:** [Topic]
**Summary:** [Brief summary]
**Action Required:** [Any follow-up needed]
```

## Tips

- Log ALL communications, even brief ones
- Note dates and method (email, phone, letter)
- Save important emails as PDFs in `original-documents/correspondence/`
- Follow up phone calls with written confirmation
