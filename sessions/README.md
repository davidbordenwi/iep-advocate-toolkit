# Sessions

Log advocacy sessions and build institutional memory.

## Purpose

Track your work across Claude Code sessions to:
- Remember what was discussed
- Capture lessons learned
- Maintain continuity
- Build a knowledge base

## Files

- `session-template.md` - Template for session logs
- Individual session logs: `YYYY-MM-DD_session.md`

## Tips

Use `/session-summary` at the end of each session to capture:
- What was accomplished
- Decisions made
- Lessons learned
- Next steps
