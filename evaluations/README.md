# Evaluations

Store evaluation summaries and analysis here.

## Contents

- **Neuropsychological evaluations** - Private evaluations
- **School evaluations** - District assessments
- **Independent Educational Evaluations (IEE)** - If obtained

## Tips

- Store original PDFs in `original-documents/evaluations/`
- Create markdown summaries here for quick reference
- Highlight key scores and recommendations
- Note any discrepancies between evaluations

## File Naming

Use format: `YYYY-MM_evaluation-type.md`
- Example: `2025-11_neuropsych-summary.md`
- Example: `2025-12_school-ot-evaluation.md`
