# Calendar

Track deadlines and reminders.

## Files

- `deadlines.md` - Upcoming deadlines with countdowns
- `reminders.md` - General reminders and recurring items

## Key IDEA Deadlines

| Deadline | Federal Requirement |
|----------|---------------------|
| Initial evaluation | 60 days from consent |
| Reevaluation | At least every 3 years |
| Annual IEP review | At least annually |
| Response to requests | Reasonable time (varies by state) |

## Tips

- Use `/check-deadlines` to view upcoming items
- Set personal reminders 1-2 weeks before deadlines
- Track school's response times
