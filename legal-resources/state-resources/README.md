# State Resources

State-specific legal resources and contacts.

## Automatic Setup

When you run `/setup`, this folder is automatically populated with your state's resources:

- **`{{state}}.md`** - Your state's comprehensive resource file
- **`template.md`** - Template used for creating state files
- **`wisconsin.md`** - Example of a fully populated state file

## What Gets Researched

The setup wizard searches for and adds:

| Resource | Source |
|----------|--------|
| Parent Training and Information Center (PTI) | parentcenterhub.org |
| Protection & Advocacy organization | ndrn.org |
| State Department of Education | Web search |
| Mediation services | Web search |
| State-specific timelines | Web search |

## Manual Updates

After automatic setup, you may want to:
- Verify contact information is current
- Add local parent support groups
- Note any recent changes in state law
- Add experiences working with these organizations

## Finding Resources Manually

- **PTI Centers:** [parentcenterhub.org/find-your-center](https://www.parentcenterhub.org/find-your-center/)
- **P&A Organizations:** [ndrn.org/about/ndrn-member-agencies](https://www.ndrn.org/about/ndrn-member-agencies/)
- **State DOE:** Search "[State] Department of Education Special Education"
