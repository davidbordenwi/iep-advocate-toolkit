# Meetings

Prepare for and document IEP meetings.

## Contents

- `templates/` - Meeting preparation and follow-up templates
- Individual meeting folders as needed

## Before a Meeting

Use `/meeting-prep` to prepare, including:
- Agenda review
- Questions to ask
- Requests to make
- Documents to bring

## After a Meeting

Use the post-meeting checklist to:
- Document what happened
- Send follow-up email
- Track action items
- File documents received
