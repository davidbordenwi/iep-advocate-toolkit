# Inbox

Drop zone for new documents to process.

## How to Use

1. Drop new documents here (PDFs, images, etc.)
2. Run `/intake` command
3. Claude will help you:
   - Identify the document type
   - Extract key information
   - File it in the right location
   - Update relevant logs

## Tips

- Process inbox regularly
- Don't let documents pile up
- Original files move to `original-documents/`
- Summaries go to appropriate folders
