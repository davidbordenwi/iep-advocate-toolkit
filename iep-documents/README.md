# IEP Documents

Store official IEP records and analysis here.

## Contents

- **Current IEP** - Most recent signed IEP
- **Previous IEPs** - Historical records
- **IEP analysis** - Your notes comparing goals, services, accommodations
- **Amendment records** - Any IEP amendments

## Tips

- Store original signed PDFs in `original-documents/iep/`
- Create markdown analysis files here
- Track changes between IEP versions
- Note any concerns or discrepancies

## File Naming

Use format: `YYYY-MM_iep-type.md`
- Example: `2025-09_annual-iep-analysis.md`
- Example: `2026-01_amendment-notes.md`
