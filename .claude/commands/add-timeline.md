# Add Timeline Entry

Add an event to the advocacy timeline log.

## Instructions

1. **Ask for event details** (if not provided):
   - Date of event
   - Type of event (meeting, phone call, email, decision, etc.)
   - Brief description
   - Key people involved
   - Outcome or next steps

2. **Format the entry** for timeline/advocacy-log.md:
   ```markdown
   ## [Date] - [Event Type]

   **Description:** [What happened]

   **People involved:** [Names and roles]

   **Outcome:** [Result or decision]

   **Next steps:** [Action items]

   **Documentation:** [Reference any saved documents]
   ```

3. **Add to timeline/advocacy-log.md** in chronological order (newest at top)

4. **Cross-reference** if relevant:
   - Update CURRENT-STATUS.md if status changed
   - Update calendar/deadlines.md if new deadlines
   - Log correspondence if communication involved

## Tips

- Be specific about dates and times
- Include names and roles of district staff
- Note any commitments made
- Document any requests or promises
