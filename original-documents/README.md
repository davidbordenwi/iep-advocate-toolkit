# Original Documents Vault

Store original PDFs and source documents here. These files are protected by .gitignore and will NOT be uploaded to GitHub.

## Folder Structure

```
original-documents/
├── evaluations/      # Neuropsych, school evals, IEEs
├── iep/              # Signed IEP documents
├── correspondence/   # Important letters, emails
├── medical/          # Medical records, doctor letters
└── school-records/   # Report cards, progress reports
```

## Important

- **NEVER commit these to git** - Contains personal information
- Keep backup copies in a secure location
- Organize by date and type
- Reference these files from markdown summaries elsewhere

## File Naming

Use format: `YYYY-MM-DD_description.pdf`
- Example: `2025-11-15_neuropsych-full-report.pdf`
- Example: `2025-09-01_signed-iep.pdf`
