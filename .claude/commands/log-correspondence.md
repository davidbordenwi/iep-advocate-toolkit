# Log Correspondence

Quick-add sent or received communications to the log.

## Instructions

1. **Ask for correspondence details** (if not provided):
   - Date
   - Direction: Sent or Received
   - Method: Email, Letter, Phone, In-person
   - From/To (name and role)
   - Subject/Topic
   - Brief summary

2. **Add entry to correspondence/correspondence-log.md:**
   ```markdown
   ## [Date] - [Sent/Received] - [Method]

   **From:** [Name, Role]
   **To:** [Name, Role]
   **Subject:** [Topic]

   **Summary:** [Brief description]

   **Action Required:** [Yes/No - if yes, what]

   **Document:** [Link to saved document if applicable]
   ```

3. **If document provided:**
   - Save to correspondence/ folder (sent or received subfolder)
   - Save original to original-documents/correspondence/

4. **Cross-reference if needed:**
   - Add to timeline if significant event
   - Update deadlines if response needed
   - Update CURRENT-STATUS.md if status changed
   - **Update MEMORY.md Session Notes if:**
     - New form/document status learned
     - Process clarification discovered
     - Correction to existing information
     - Important context for future sessions

## Quick Log Format

For rapid logging, user can provide:
```
[date] [sent/received] [email/call/letter] [person] - [topic]
```

Example: "1/15 received email Mrs. Smith - consent form sent"
