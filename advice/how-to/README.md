# How-To Guides

Step-by-step procedures for common tasks.

## Topics

Create guides for procedures like:
- How to request an evaluation
- How to request an IEP meeting
- How to get records (FERPA request)
- How to request an IEE
- How to file a complaint

## Format

Each topic has:
- `topic-quick.md` - One-page quick reference
- `topic-detailed.md` - Full guide with examples
