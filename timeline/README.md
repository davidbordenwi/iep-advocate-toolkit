# Timeline

Chronological log of all advocacy events.

## Files

- `advocacy-log.md` - Main chronological log

## Using the Log

Use `/add-timeline` to add entries, or add manually:

```markdown
### YYYY-MM-DD - Event Type
**Type:** Meeting/Communication/Deadline/Decision/Document
**Summary:** [What happened]
**Outcome:** [Result]
**Next Steps:** [Follow-up needed]
```

## Tips

- Log everything, even small events
- Be specific about dates
- Note outcomes and next steps
- Reference documents stored elsewhere
