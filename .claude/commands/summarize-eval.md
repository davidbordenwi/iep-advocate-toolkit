# Summarize Evaluation

Summarize an evaluation document and extract key findings.

## Instructions

1. **Ask which evaluation to summarize** (if not provided):
   - Check evaluations/ folder for existing summaries
   - Check inbox/ for new documents to process
   - Or have user provide the document path

2. **Read the evaluation document**

3. **Create a summary** including:
   - Evaluation type and date
   - Evaluator name and credentials
   - Key standardized scores (with percentiles)
   - Strengths identified
   - Areas of concern/weakness
   - Specific recommendations
   - Eligibility implications

4. **Format the summary** for evaluations/ folder:
   ```markdown
   # [Evaluation Type] Summary

   **Date:** [Date]
   **Evaluator:** [Name, Credentials]
   **Type:** [Neuropsych, OT, Speech, Educational, etc.]

   ## Key Scores
   | Assessment | Score | Percentile | Notes |
   |------------|-------|------------|-------|
   | [Test name] | [score] | [percentile] | [interpretation] |

   ## Strengths
   - [Strength 1]
   - [Strength 2]

   ## Areas of Concern
   - [Concern 1]
   - [Concern 2]

   ## Recommendations
   - [Recommendation 1]
   - [Recommendation 2]

   ## Implications for IEP
   [Analysis of how findings affect services]
   ```

5. **Save to evaluations/** folder with descriptive name

6. **Update STUDENT.md** Key Data Points section if this evaluation provides new baseline data

## Tips

- Focus on scores below 25th percentile (potential service triggers)
- Note any discrepancies between scores
- Highlight recommendations that align with IDEA services
