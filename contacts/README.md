# Contacts

Store contact information for everyone involved.

## Files

Create contact files for:
- `school-contacts.md` - Teachers, principal, case manager
- `iep-team.md` - All IEP team members
- `medical.md` - Doctors, therapists, evaluators
- `support.md` - Advocates, attorneys, parent groups

## Tips

- Keep contact info current
- Note best times/methods to reach people
- Track role changes (staff turnover is common)
- Include support organizations in your state
